# ReactMetricsDashboad-for-Monitoring-MobileApp
Récupérer des données depuis une API distante
Dans des cas d'utilisation réels, les points de terminaison d'API distants fournissent les métriques pour les applications sur la base desquelles vous pouvez afficher un tableau de bord en fonction des exigences commerciales.

Certaines des métriques d'application les plus utiles incluent :

Utilisation quotidienne
Données hebdomadaires sur un mois
Utilisation par catégorie, comme les réseaux sociaux, la productivité, le divertissement, etc.
Utilisation par fonctionnalités de l'application telles que la connexion, le paiement, le chat, etc.
